Titre
		générer des cases spéciales
Résumé
		le jeu génère des cases spéciales de manière aléatoire
Auteur
		Diana
Responsable
		Diana
Date de création
		09/10/2018
Acteur concerné
		le jeu
Pré-condition
		le labyrinthe est généré
Post-condition
		 une case spéciale apparaît sur le labyrinthe
Scénario normal
		1. le jeu choisit le nombre de cases spéciales
		2. le jeu choisit un type de case à poser sur le labyrinthe
		3. le jeu choisit la position des cases spéciales sur le labyrinthe

scénario exceptionnel
		E1. aucune case n'est à générer
